Xpdite

An AI assistant and agent harness that is truly yours. It runs your desktop using any model provider, is completely open-source, 100% customizable, and built for everyone.

Use it as a desktop copilot, then continue the same experience from mobile channels through WhatsApp, Telegram, or Discord.

What Xpdite Helps Users Do

  • Summarize Gmail conversations and extract key action items.
  • Create, update, and organize Google Calendar events from natural language requests.
  • Run agent-style, multi-step workflows with approval-aware actions.
  • Maintain desktop and mobile continuity through WhatsApp, Telegram, and Discord channels.

Local-First by Design

Xpdite runs on the user's own computer. Core processing and data usage are local to the user, and legal details for Google API usage are described in the linked policies.

For compliance details, see the Privacy Policy and Terms of Service.
