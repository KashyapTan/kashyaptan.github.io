Privacy Policy

Effective date: April 20, 2026

This Privacy Policy explains how Xpdite handles data when users connect Google services. Xpdite is a desktop app that runs locally on the user's own computer.

Data Collection and Use

Xpdite accesses Google User Data, including Gmail messages and Google Calendar events, only to provide requested features such as email summarization and calendar scheduling.

Xpdite runs 100% locally on the user's own device, so Gmail and Calendar content is not centrally collected or stored by Xpdite for normal app usage.

User data remains on the user's device unless the user explicitly chooses to share information for support troubleshooting or a security investigation.

Google API Limited Use Disclosure

Xpdite’s use and transfer of information received from Google APIs to any other app will adhere to Google API Services User Data Policy, including the Limited Use requirements.

Privacy Commitments

  • Xpdite does not sell user data to third parties.
  • Xpdite does not use Google user data for serving advertisements.
  • No humans, including developers, read user Gmail or Calendar data unless the user gives explicit consent for troubleshooting or a security investigation.
  • Xpdite is designed as a local-first desktop application, so data processing is performed on the user's machine whenever possible.

Contact

Questions about this policy can be sent through the contact form on kashyaptan.com.
