
# Personal Portfolio Website

This website highlights my background, projects, and skills. It also includes a contact page with links to my LinkedIn, email, and GitHub.


## Features/Capabilities/Remarks
- Custom hamburger menu that pops in and out on click. 
- Smooth Scroll.
- GSAP Animations.
    - Word specific animations.
    - Horizontal Scrolling.
    - Pin page animations.
    - General scrollTrigger animations. 
- Custom made and designed elements and logos using Figma.
- Contact page with inbuilt email sender.

## Xpdite

Xpdite is an open-source AI desktop assistant and agent harness.

- Added a dedicated `Xpdite/` landing page to explain what it is and what it helps users do.
- Includes simple install commands for macOS and Windows.
- Supports local-first usage and links to legal pages (`/Xpdite/privacy` and `/Xpdite/tos`).
- Focused on helping with workflows like Gmail summaries, Google Calendar actions, and multi-step agent tasks.
